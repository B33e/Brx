¿Quiénes somos?

Wash.WEB es un aplicativo para mejorar el servicio de recepcion de clientes en un autolavado con el propósito de brindar un servicio más seguro y cómodo para el cliente.

En este sistema está el administrador que es el encargado de registrar y asignar los trabajadores con sus respectivos perfiles, también es el encargado de organizar los horarios de las citas para que este sea un sistema muy ordenado. En la aplicación aparecerán precios de el lavado y el tiempo que se demora el lavado, localización de el autolavado por GPS.

Introducción

El objetivo de este proyecto es crear un aplicativo funcional el cual permita mejorar la recepcion en un autolavado, y a su vez, aumentar la comodidad del cliente a la hora de agendar una cita mejorando aspectos de sguridad en el proceso evitando que se pierda el orden por motivos de soborno.

Problema y justificación

El problema que se busca solucionar es la la falta de orden en un lavadero de vehiculos lo que hace que este sistema se vuelva lento, desordenado y en los peores casos poco seguro para el estado del vehiculo, nuestro aplicativo se utilizara en un lavadero en especifico con el proposito de que el lavadero aumente las medidas de organizacion ademas de mejorar los estandares de seguridad de los vehivulos.

Objetivos

General

Crear una app con el propósito de aumentar la comodidad en la recepción de un autolavado y aumentar medidas de seguridad.

Específicos

  • Mejorar la seguridad de un autolavado.
  • Mostrar la ubicación del autolavado.
  • Agendar una cita con el trabajador del autolavado.
